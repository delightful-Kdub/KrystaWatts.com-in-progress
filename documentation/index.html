Home

Home is powered by the amazing jQuery plugin called ISOTOPE.Isotope works by placing elements in optimal position based on available vertical space, sort of like a mason fitting stones in a wall.

Journal

Contact

Proton contact form works with php form, to edit it you need to go includes/form.php, everything there is commented for any edit, In second line you can change the email adress and in the third line the message.
